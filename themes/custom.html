--- layout: page title: Build your own theme ---

Rolling your own theme is easy!

Custom build

Once you have a custom build on your machine you are ready to begin.

Creating themes

Don't change any of the source code; all your theme code will be new and will live under the /scss/themes folder.

Step 1
Create file

Create a new file e.g. /scss/themes/blaze.your-theme-name-here.scss

Step 2
Variables

Copy the following into your newly created file: 
@import "../mixins/settings.global";

$border-radius: 0;
/* ... add more variables here ... */

.t-your-theme-name-here {
  @import "../blaze";

  /* ... add class overrides here ... */
}

Step 3
Build theme

From your command line npm run build
Once complete your new themed version of Blaze will appear in the /dist/themes folder, as well as inside the demo folder.

Step 4
Test it out

Edit /demo/themes/index.html and replace the default Blaze stylesheet link yours, and add your theme class to the HTML element <html class="c-your-theme-name">...</html>.

Now open in your browser and see your new theme!

Submitting your themes

If you would like to share your theme with others please submit a Pull Request.

Fork Blaze, make your theme and submit a Pull Request to our dev branch for review.

If it is successfully submitted we will add it to our Showcase!
