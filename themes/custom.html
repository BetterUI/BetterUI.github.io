Rolling your own theme is easy!

Custom build

Once you have a custom build on your machine you are ready to begin.

Creating themes

Don't change any of the source code; all your theme code will be new and will live under the /scss/themes folder.

Step 1
Create

The following command will create a new theme file
gulp theme --name <your-theme-name>

Step 2
Change

Open the file, add any variables you want to override plus any additional code your theme needs.

Step 3
Build

Build your new theme, the output will go into /dist/themes and /demo/themes
gulp

Step 4
Test

Edit /demo/index.html and replace the default Blaze stylesheet link with yours. 
<link rel="stylesheet" href="themes/blaze.<your-theme-name>.min.css">
Open it in your browser and see your new theme!

Submitting your themes

If you would like to share your theme with others please submit a Pull Request.

Fork Blaze, make your theme and submit a Pull Request to our dev branch for review.

If it is successfully submitted we will add it to our Showcase!
