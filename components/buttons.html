Probably the first thing people look at when choosing a CSS framework. The common button!

Basic Button

To make a button a Blaze button add the .button class to buttons, inputs or links.

Link 
<button class="button">Button</button>
<input type="button" class="button" value="Input">
<a class="button">Link</a>
<button class="button">Button</button>

Button Colours

There are four button colour modifiers .button--primary, .button--secondary, .button--success and .button--error. 

<button class="button">Button</button>
<button class="button button--primary">Button</button>
<button class="button button--success">Button</button>
<button class="button button--error">Button</button>

Button Group

To group related buttons together you can use the .button-group wrapping element. This will bunch the buttons together and remove the spacing between them. 

<span class="button-group">
  <button class="button button--primary">Button</button>
  <button class="button button--secondary">Button</button>
</span>
