From single input controls like text boxes to full form patterns.

Add .field to your text boxes.

Text Input

<input class="field" type="text">
<textarea class="field"></textarea>

Colours

<input class="field field--success" type="text">
<input class="field field--error" type="text">
<input class="field" disabled type="text">

Sizes

<input class="field field--xsmall" type="text">
<input class="field field--small" type="text">
<input class="field field--medium" type="text">
<input class="field field--large" type="text">
<input class="field field--xlarge" type="text">
<input class="field field--super" type="text">

If you want to squash a controls together...

  <textarea class="field"></textarea>

  <input class="field">
  <textarea class="field"></textarea>
  <input class="field">
</div>

  <input class="field" type="text">
  <button class="button button--primary">Action</button>
</div>
<div class="input-group">
  <button class="button button--secondary">Action</button>

  <button class="button button--primary">Action</button>
</div>
<div class="input-group">

</div>

Fieldsets and Legends

Use .fieldset and .fieldset__legend classes to apply Blaze's styles to ensure consistent fieldset behaviours.

A group of controls 
<fieldset class="fieldset">
  <legend class="fieldset__legend">A group of controls</legend>
  <label class="choice">
    <input type="radio" name="optionsRadios"> Option one
  </label>
  <label class="choice">
    <input type="radio" name="optionsRadios"> Option two
  </label>
</fieldset>

Disabled Fieldset

Disabling a fieldset will disable all controls within it and Blaze will apply the disabled styling.

A group of controls 
<fieldset class="fieldset" disabled>
  <legend class="fieldset__legend">A group of controls</legend>
  <label class="choice">
    <input type="radio" name="optionsRadios"> Option one
  </label>
  <label class="choice">
    <input type="radio" name="optionsRadios"> Option two
  </label>
</fieldset>

Labels and Form Elements

You can either use the for attribute on labels to tie the label to the input or wrap your fields within labels. If you wrap your fields within labels replace the .field class with .label__field element class on your text inputs.

We recommend wrapping your label and inputs within a .form-element block, this will supply the block with some vertical padding; ready to be placed within a form. 

<!-- Approach one -->
<div class="form-element">
  <label class="label" for="nickname">Nickname:</label>
  <input id="nickname" class="field" placeholder="Nickname" >
</div>
<!-- Approach two -->
<label class="label form-element">
  Real name:
  <input class="label__field" placeholder="Real name">
</label>

Forms

Blaze has no classes for forms, rather it relies on styling the child blocks of forms; fieldsets, legends, form-elements, labels, fields and choices.

Please complete the form:
Account login: 
<fieldset class="fieldset">
  <legend class="fieldset__legend">Please complete the form:</legend>
  <div class="form-element">
    <label class="label" for="houseno">
      Address line 1:
    </label>
    <input id="houseno" placeholder="House name or number" class="field">
  </div>
  <label class="label form-element">
    Postcode:
    <input class="label__field" placeholder="e.g. AB12 3CD">
  </label>
  <label class="label choice">
    <input type="checkbox">
    Send me spam
  </label>
  <fieldset class="fieldset list list--inline list--unstyled">
    <legend class="fieldset__legend">Account login:</legend>
    <label class="label choice list__item">
      <input type="radio" name="account" checked>
      New user
    </label>
    <label class="label choice list__item">
      <input type="radio" name="account">
      Existing user
    </label>
  </fieldset>
</fieldset>
